搜索二维矩阵 II

编写一个高效的算法来搜索 m x n 矩阵 matrix 中的一个目标值 target 。该矩阵具有以下特性：

每行的元素从左到右升序排列。 每列的元素从上到下升序排列。  

示例 1：

输入：matrix = [[1,4,7,11,15],[2,5,8,12,19],[3,6,9,16,22],[10,13,14,17,24],[18,21,23,26,30]], target = 5 输出：true

分析：

z字形法

从(0, n-1)开始，如果当前值比target大，说明这一列都比target大，可以舍弃这一列，y-=1

同理，如果当前值比target小，说明这一行都比target大，可以舍弃这一行，x+=1

无重叠区间

给定一个区间的集合 intervals ，其中 intervals[i] = [starti, endi] 。返回 需要移除区间的最小数量，使剩余区间互不重叠 。

示例 1:

输入: intervals = [[1,2],[2,3],[3,4],[1,3]] 输出: 1 解释: 移除 [1,3] 后，剩下的区间没有重叠。 示例 2:

输入: intervals = [ [1,2], [1,2], [1,2] ] 输出: 2 解释: 你需要移除两个 [1,2] 来使剩下的区间没有重叠。 示例 3:

输入: intervals = [ [1,2], [2,3] ] 输出: 0 解释: 你不需要移除任何区间，因为它们已经是无重叠的了。

分析：

无

分析：

经典的贪心问题

关键点1：左排序or右排序

此题与会议问题相似，在多个会议时间中如何安排能够实现最多会议

那么首先就要保证第一个会议尽早结束，这样才留着其他会议更多的时间分配，所以选择右排序

关键点2：如何比较

右排序后，谁和谁比较呢？

第一，确定首个区间后，我们的目的是找到下一个不与其交集的区间

所以选择intervals[i][0] >= right，这里的right代表的是上一个被确定的不相交的区间的右端点intervals[i][1]，一旦上述成立，说明我们又找到一个不相交的区间了，ans+=1， right = intervals[i][1]

最后遍历完成之后ans就代表了最大不相交的区间数

用数组长度减去ans即为最后结果
